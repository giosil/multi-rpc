# Multi-RPC - WDemo - Backend

## Build

- `mvn clean package`

## Build Docker Image

- `docker build -t wdemo-be .`

or

- `docker build -t wdemo-be:1.0.0 .`
- `docker tag wdemo-be:1.0.0 wdemo-be:latest`

## Remove Docker Image

- `docker rmi wdemo-be`
